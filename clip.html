The beginning of the current contents of clipboard is displayed in this window.

This clipboard is tightly tied to the Windows clipboard. Whenever this clipboard changes, Windows clipboard is also updated. Whenever Windows clipboard changes, this clipboard is updated (when you are searching characters by clipboard or searching words by clipboard, the windows are automatically refreshed).

Note: Program can handle only UNICODE clipboard format. If you want to paste some data from this program make sure the application supports UNICODE.

Window controls

  • - Clears clipboard contents.
